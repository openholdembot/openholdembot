PokerTracker Symbols

OpenHoldem supports PokerTracker symbols, and so does OpenPPL. The development team did already create some symbols of general use that get shipped with the library. However: the possibilities are endless and you can for sure imagine other symbols that are useful. If you want to increase the power of OpenPPL please have a look at the chapter “Building Symbols On Your Own” and at Openholdems manual. It is not that difficult and if you want to contribute to the project your symbols could even become part of the next version.
Note: The PokerTracker symbols need a working connection to Poker Tracker Database (PT2/PT3 depending on OH version), correct opponents name scraping and sitename/network symbols in the TableMap as described in the OpenHoldem manual.
All PokerTracker symbols require a certain samplesize to become meaningful. Our library defines reasonable requirements for the stats and returns -1 if there are not enough hands.
Please note that OH returns percentages in the range [0..1], so 0.31 means 31%.
Invalid value return -1. You need to remember that if you check any of those symbols as “less then” of a certain value and the returned value is -1 you need to add a sanity check.
Symbol Meaning
PT_BigBlind_FoldBBToSteal Big Blind fold percentage to steal-raises from late position or from the small blind
PT_ButtonVPIP Button VPIP
PT_BigBlind_Hands Big Blind fold percentage to steal-raises from late positions
PT_LastCaller_VPIP Last caller percentange of voluntary played hands
PT_LastCaller_PFR Last caller percentange of preflop raise
PT_LastRaiser_AttemptToStealBlinds Percentage of steal-raises when first-in and in late position or in the small-blind
PT_LastRaiser_Hands Last Raiser player number of hands stored in PT database
PT_LastRaiser_PFR Preflop raising percentage of the last raiser
PT_LastRaiser_VPIP Percentage of volutary played hands by the last raiser
PT_OpponentVPIPHeadsUp VPIP of opponent when playing heads up
PT_SmallBlind_FoldSBToSteal Small Blind fold percentage to steal-raises from late positions
PT_SmallBlind_Hands Player in Small Blind number of hands stored in PT database
PT_SmallBlind_VPIP Percentage of voluntary played hands by the small blind
