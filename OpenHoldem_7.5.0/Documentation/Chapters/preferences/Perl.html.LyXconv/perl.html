OpenHoldem Preferences – ↓Perl

Perl preferences control how OpenHoldem interacts with the installed Perl interpreter and your Perl formulas. See the {Perl} section for more information on installing and configuring Perl.
  • Default formula: This is the path to the Perl formula that will be automatically loaded on startup if the ActiveState perl interpreter is correctly installed. Just leave this option empty if you don’t intend to use Perl.
  • Perl editor: This is the path to the editor that will be launched when a Perl formula is selected for editing from the Perl menu on the main window.
