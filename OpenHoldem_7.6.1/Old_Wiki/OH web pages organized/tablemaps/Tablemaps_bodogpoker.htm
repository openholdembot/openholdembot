[edit] Bodog Poker

NOTE: Would like to add that Bodog is changing his software completely so all active work now can be quit I guess


The Bodog Poker client under Windows XP and Vista have 'fuzzy' (anti-alias) fonts that can cause problems with scraping. Support for fuzzy text is currently under development in OpenHoldem, it is scheduled to be introduced in OpenScrape v1.1 and OpenHoldem v1.6.

The table map currently being developed is for fuzzy text.

You can run Bodog Poker under Windows 2000 to avoid fuzzy text.
