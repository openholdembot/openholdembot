[edit] Project Files

The OpenHoldem source archive contains a number of different projects:

  • OpenHoldem x.x.x is the main OpenHoldem application
  • poker-eval is a cut down version of the open source poker-eval library
  • runron_table is used to create the arrays that are found at the top of the runron.cpp file in the OpenHoldem x.x.x project
  • Scilexer is the open source editor component used in the OpenHoldem x.x.x Formula Editor dialog
  • test dll is a sample project to be used for creating an OpenHoldem compliant DLL extension
  • versus_table is used to create the versus.bin file
