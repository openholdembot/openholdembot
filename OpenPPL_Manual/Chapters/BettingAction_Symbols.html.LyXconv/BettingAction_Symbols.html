BettingAction Symbols

Name

Meaning

Limitations

Bets

The number of bets made this betting round, can only be 0 or 1 since the 2nd bet is a raise

None

BotCalledBeforeFlop

We called preflop, can also be used on the flop and will be true if we called

None

BotIsLastRaiser

We have the betting / raising initiative

None

BotRaisedBeforeFlop

We raised preflop, can also be used on the preflop and will be true if we bet or raised

None

BotRaisedOnFlop

We bet or raised on the flop, can also be used on the flop and will be true if we bet or raised

Postflop only

BotRaisedOnTurn

We bet or raised on the turn, can also be used on the turn and will be true if we bet or raised

Turn and river only

BotsActionsOnFlop

Number of actions where we put money in the pot on the flop

All-ins are not counted obviously (as then the game would be over anyway)

BotsActionsOnThisRound

Number of actions this betting round where we put money in the pot

All-ins are not counted obviously (as then the game would be over anyway)

BotsActionsOnThisRoundIncludingChecks

Number of checks + number of actions where we put money in the pot

All-ins are not counted obviously (as then the game would be over anyway)

BotsActionsPreflop

Number of actions where we put money in the pot preflop, all-ins are not counted

None

BotsLastAction

Bots last action, can be one of the following: none, beep, raise, bet, call, or check

None

BotsLastPreflopAction

Bots last action preflop, can be one of the following: none, beep, raise, bet, call, or check

None

CalledOnFlop

We called on the flop, may also be used on the flop and will be true if we called

Postflop only

CalledOnTurn

We called on the turn, may also be used on the turn and will be true if we called

Turn and river only

Calls

The number of calls by opponents on this betting round

None

CallsSinceLastRaise

The number of calls by all opponents since the last raise by an opponent on the current betting round

None

Checks

The number of checks made by opponents this betting round

None

Folds

The number of folds this betting round

None

NoBettingOnFlop

No bets/raises were made on the flop, may also be used on the flop and will be true if there were no bets/raises

Postflop only

NoBettingOnTurn

No bets/raises were made on the turn, may also be used on the turn and will be true if there were no bets/raises

None

NumberOfRaisesBeforeFlop

The number of raises before the flop made by opponents

None

NumberOfRaisesOnFlop

The number of raises on the flop made by opponents

Postflop only

NumberOfRaisesOnTurn

The number of raises on the turn made by opponents

Turn and river only

OpponentCalledOnFlop

An opponent called on flop, and did not raise or bet

Postflop only

OpponentCalledOnTurn

An opponent called on turn, and did not raise or bet

Turn and river only

OpponentIsAllin

An opponent is all in, there may still be other players in the hand

None

Raises

The number of raises made by opponents this betting round

None

RaisesBeforeFlop

An opponent raised before flop

None

RaisesOnFlop

An opponent raised on flop

Postflop only

RaisesOnTurn

An opponent raised on turn

Turn and river only

RaisesSinceLastPlay

The number of raises since our last action

None
