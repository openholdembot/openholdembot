REMARK. If a set is on board, “PairOnBoard” is also true; full houses do also count as sets and psirs, made flushes as flushdraws, etc. Therefore it is a strongly recommended to code made hands first (and strong made hands at the very beginning), then weaker holding with positive potential and weaker draws at the vera last.

NutFullHouseOrFourOfAKind

This symbol evaluates the strenght of quads and full houses. Top quads get always rated as 1, bottom quads or best-full house as 2, next best hands as 3 and so on. This symbol does not take straight flushes into account. So it could return 1 even if our hand can be beaten by a straight flush.

Postflop only.

REMARK 1. Standard PPL returns 0 if we don’t have any quads / FH at all. However we think this is counterintuitive and causes problems, as a smaller number means a better hand. OpenPPL returns 999 for that case.
