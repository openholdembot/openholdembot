Toolbars

OpenHoldem Main Toolbar


Figure 0.0.1: OpenHoldem Main Toolbar

From left to right, the buttons on the Main toolbar are:

  • New formula file
  • Open formula file
  • Save formula file
  • Connect to table (green circle)
  • Disconnect from table (red circle)
  • Engage the Autoplayer (nested diamonds)
  • Edit formula
  • Enable validator
  • Show/hide table display
  • Attach OpenHoldem window to the top of the table
  • Attach OpenHoldem window to the bottom of the table
  • Lock blinds
  • Show scraper output
  • Shoot a replay-frame
  • Show the help-file (this file!)
