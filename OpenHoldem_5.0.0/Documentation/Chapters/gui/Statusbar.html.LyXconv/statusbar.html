Status Bar


Figure 0.0.1: OpenHoldem Statusbar

From left to right, the fields on the Status Bar are:

  • Your hole cards
  • The board cards
  • Your poker hand
  • Handrank
  • PrWin, PrTie and PrLos (the values mean: winning-percentages * 10)
  • Current value of f$P, one of the standard formula functions. This formula is is intended to specify the number of opponents used for the prwin-calculations; however this number can be more or less than the number of opponents actually seen on the table at any given time. More information on f$P can be found section about Primary Functions and in the section about PrWin.
  • Number of iterations done and number of iterations to be done
  • Recommended action
