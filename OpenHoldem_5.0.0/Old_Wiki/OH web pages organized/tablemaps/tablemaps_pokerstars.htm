[edit] Settings

Note: For version Stars.9.0.0.4 "Display Bet Amounts" must also be selected. Also, for version 9.0.0.4 all chips except the dealer chip must be renamed to the form cxhip. A whole bunch of files in folders named 0 - 6 must be renamed in the PokerStars gfx\chips folder. Remember to not rename chip-d.bmp and chip-d.a.bmp.


Source(s):Poker Stars

[edit] Know Issues

Limit ring game. If the raises are capped, the call button will appear in the raise button location. The raise button will be invisible. Thus, OH thinks only fold/raise buttons are available. This will cause the call/check action result in fold action. Note: This issue hasn't been tested on table map version 9.0.0.4.
