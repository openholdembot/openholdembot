Commercial DLLs

Back in the old days of WinHoldem there have been mainly two commercial DLLs to extend the functionality of the framework: Bozos DLL (www.bozobot.net) and SingleMalts DLL (now available as open source at http://code.google.com/p/singlemalts-whuser-dll/). Both did provide additional symbols like PokerTracker stats, memory and short-circuiting.

Bozos DLL does not work with OpenHoldem and the functionality of SingleMalts DLL got completely integrated into OpenHoldem, so there is no need to use any one of them. Openholdem will likely crash!
